# Mynther - Fashion E-commerce

This is a modern React + Vite application for a fashion e-commerce platform.

## How to Run Locally

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Setup API Key**
    Create a file named `.env` in the root directory and add your Google Gemini API Key:
    ```env
    API_KEY=your_actual_api_key_here
    ```

3.  **Run the App**
    ```bash
    npm run dev
    ```
    Open the URL provided (usually http://localhost:5173).

## Features
- 280+ Products across 7 categories
- AI Stylist Chat
- Profile/Login System (Demo)
- Cart & Checkout Flow
- Search & Filtering
